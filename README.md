# Leisure Shift Flow

A shift-scheduling app with a React frontend and a Node/Express backend.

## Project structure

```
backend/    Express API server
frontend/   React app
```

## Getting started

### Backend

```
cd backend
npm install
npm start
```

The API server starts on `http://localhost:4000`.

### Frontend

```
cd frontend
npm install
npm start
```

The React app starts on `http://localhost:3000`.
